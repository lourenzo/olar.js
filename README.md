# olar.js
A LMFAO semantic library for JS. Just for fun.

## Assertion:

```js
const olar = require('olar.js');

olar.
    Eu.te.quero.para.jesus( false == -1 );
```


## Persistence:

```js
const olar = require('olar.js');

const sampleData = {
    jogo: 'dotA'
}

olar.
    Que.deus.te.guarda({ ...sampleData });

```

## Queries:

```js
const olar = require('olar.js');

olar.
    Cade.vc({ name: /Rosilandi/ });

```
